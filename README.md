# 📦 FedEx Supply Chain EDA

# Project Overview

This project performs Exploratory Data Analysis (EDA) on the FedEx Supply Chain dataset to identify trends, patterns, and business insights related to shipments, vendors, transportation, inventory, and logistics costs.

The goal is to transform raw supply chain data into meaningful insights that help improve operational efficiency and support data-driven business decisions.

---

# Business Objective

The objective of this project is to:

- Understand shipment distribution across countries.
- Analyze vendor and product performance.
- Identify missing values and data quality issues.
- Study freight, insurance, and shipment costs.
- Discover patterns that can improve logistics and supply chain operations.
- Provide actionable business recommendations.

---

# Dataset Information

- Dataset: FedEx Supply Chain Dataset
- Records: 10,000+ rows
- Features: 33 Columns

The dataset contains information related to:

- Countries
- Vendors
- Products
- Shipment Modes
- Delivery Dates
- Freight Cost
- Insurance Cost
- Product Categories
- Manufacturing Sites
- Shipment Quantity
- Shipment Value

# Key Insights

- Shipment distribution varies significantly across countries.
- A few vendors handle most shipments.
- Air shipment mode is frequently used.
- Some product categories contribute more to shipment volume.
- Freight cost increases with shipment value.
- Missing values were found in Dosage, Shipment Mode, and Insurance columns.
- Most records are unique with no duplicate rows.

---

# Business Recommendations

- Improve data quality by reducing missing values.
- Optimize logistics using efficient shipment modes.
- Strengthen relationships with high-performing vendors.
- Monitor freight and insurance costs regularly.
- Improve inventory planning for high-demand products.
- Develop dashboards for real-time shipment monitoring.
